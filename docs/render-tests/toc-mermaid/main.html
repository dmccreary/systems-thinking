Causal Loop Diagram: Tragedy of the Commons Archetype

This causal loop diagram (CLD) illustrates the "Tragedy of the Commons" systems archetype using Mermaid syntax. It depicts two actors (A and B) utilizing a shared resource, with reinforcing loops (R1 and R2) driving individual gains and a shared balancing loop (B) representing resource depletion, which ultimately reduces gains for all.

graph LR A["A's Activity"] -->|+| AR["Net Gains for A"] AR -->|+| A B["B's Activity"] -->|+| BR["Net Gains for B"] BR -->|+| B A -->|+| T["Total Activity"] B -->|+| T T -->|-| G["Gain per Activity"] G -->|+| AR G -->|+| BR subgraph "R1 (Reinforcing Loop for A)" A -- + --> AR -- + --> A end subgraph "R2 (Reinforcing Loop for B)" B -- + --> BR -- + --> B end subgraph "B (Balancing Loop)" A -- + --> T -- - --> G -- + --> AR -- + --> A end

Explanation:

  • Reinforcing Loops (R1 and R2): Each actor increases their activity to gain more, creating a virtuous cycle for the individual.
  • Balancing Loop (B): Total activity depletes the shared resource, reducing the gain per activity, which eventually limits gains for all, though individuals don't immediately feel the full impact.
  • Polarities: '+' indicates a positive (same direction) causal relationship, '-' indicates a negative (opposite direction) relationship.

This archetype demonstrates how rational individual actions can lead to collective ruin in shared resource systems, such as overfishing, deforestation, or pollution.
